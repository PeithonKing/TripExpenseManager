Trip Expense Manager

Split shared trip costs and generate fewer settlement payments.

How to Use

Step 1: Add People

Click Add Person and enter names of everyone in the trip. In case the name is not reflected immediately, don't panic, add all the names, or add a new row, and it will show up.

Step 2: Add Expense Rows

Click Add Row for each expense entry.

Step 3: Fill Each Row

  • Provide expense name and total amount spent.
  • Select who paid using Spent By.
  • Enter share values for each person using ratio-style values (like 1, 2, 0.5) or direct amounts.

Step 4: Calculate

Click Calculate to see payment summary, settlement transactions, and individual totals.
